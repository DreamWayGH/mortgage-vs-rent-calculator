# 房貸 vs 租房+投資 比較計算機 — 需求規格文件

## 一、專案概述

| 項目 | 說明 |
|------|------|
| 專案名稱 | 房貸 vs 租房+投資 比較計算機 |
| 目的 | 讓使用者快速試算不同購屋/租房情境下的財務結果，並可並列比較多組參數 |
| 技術方案 | React框架，使用mui、tailwindcss，瀏覽器開啟即用，無需後端 |
| 目標使用者 | 有購屋/租屋決策需求的一般民眾 |

## 三、計算邏輯

### 3.1 房貸計算

1. **自備款** = 房屋總價 ×（1 - 貸款成數）
2. **總貸款金額** = 房屋總價 × 貸款成數
3. **新青安貸款額** = MIN（總貸款金額, 新青安額度上限）
4. **超額貸款額** = 總貸款金額 - 新青安貸款額（若 ≤ 0 則為 0）
5. **寬限期月繳** = 新青安貸款額 × 新青安月利率 + 超額貸款額 × 超額月利率
6. **攤還期年數** = 貸款年限 - 寬限期
7. **攤還期月繳** = 兩筆貸款分別以「本息均攤公式」計算後加總
   - 本息均攤公式：`P × r × (1+r)^n / [(1+r)^n - 1]`
   - P = 本金, r = 月利率, n = 攤還月數
8. **總繳款** = 寬限期月繳 × 寬限月數 + 攤還期月繳 × 攤還月數
9. **總利息** = 總繳款 - 總貸款金額
10. **N 年後房產市值** = 房屋總價 × (1 + 房價年漲幅)^N

### 3.2 租房 + 投資計算

1. **自備款全額投入 ETF**，以年化報酬率複利滾存
2. 每月差額 =（買房月繳 + 每月持有成本）- 當期月租金
   - 差額 > 0（買房總支出較高）：租房方將差額投入 ETF
   - 差額 < 0（月租金較高）：買房方將差額絕對值投入 ETF
3. **租金隨年漲幅逐年調整**，每月差額隨之變動
4. **租房方 ETF 累積資產** = 自備款複利 + 差額 > 0 月份的定期定額未來值（需逐月計算，因租金每年變動）
5. **買房方 ETF 累積資產** = 差額 < 0 月份的定期定額未來值（需逐月計算）
6. **租房總支出** = 所有月份租金加總
7. **累計持有成本** = 每月持有成本 × 12 × 貸款年限

### 3.3 房地合一稅計算（自住優惠）

1. **課稅所得** = N 年後房產市值 - 房屋總價（即增值部分）
2. 稅額計算：
   - 持有 ≥ 6 年（自住）：課稅所得 ≤ 400 萬免稅，超出部分 × 10%
   - 持有 2～5 年：課稅所得 × 35%
   - 持有 < 2 年：課稅所得 × 45%
3. 若課稅所得 ≤ 0（房價未增值），稅額 = 0
4. 本計算器預設持有年限 = 貸款年限（通常 ≥ 6 年，適用自住優惠）

### 3.4 比較結果（N = 貸款年限）

1. **買房淨資產** = N 年後房產市值 - 房地合一稅 + 買房方 ETF 累積資產
2. **租房淨資產** = 租房方 ETF 累積資產
3. **淨資產差距** = 租房淨資產 - 買房淨資產

## 四、輸出規格

### 4.1 比較總表（主表格）

每次按「計算」新增一列，欄位如下：

| 欄位 | 來源 |
|------|------|
| 序號 # | 自動遞增 |
| 房屋總價 | 輸入值 |
| 貸款年限 | 輸入值 |
| 寬限期 | 輸入值 |
| 新青安利率 | 輸入值 |
| 月租金 | 輸入值 |
| 租金漲幅 | 輸入值 |
| ETF 報酬率 | 輸入值 |
| 房價漲幅 | 輸入值 |
| 寬限期月繳 | 計算 |
| 攤還期月繳 | 計算 |
| 總利息（萬） | 計算 |
| N 年後房產市值（萬） | 計算 |
| 買房方 ETF 資產（萬） | 計算 |
| 租房方 ETF 資產（萬） | 計算 |
| 累計持有成本（萬） | 計算：每月持有成本 × 12 × 貸款年限 ÷ 10000 |
| 房地合一稅（萬） | 計算：依 3.3 節自住優惠公式 |
| 買房淨資產（萬） | 計算：房產市值 - 房地合一稅 + 買房方 ETF |
| 租房淨資產（萬） | 計算：租房方 ETF |
| 淨資產差距（萬） | 計算：租房淨資產 - 買房淨資產 |
| 勝出方 | 標示「買房勝」或「租房+投資勝」 |
| 操作 | [刪除] 按鈕 / [展開明細] 按鈕 |

### 4.2 展開明細（子表格）

點擊某列的「展開明細」後，在該列下方展開子表格：

| 欄位 | 說明 |
|------|------|
| 年份 | 每 5 年一列（第5年、第10年…到貸款年限） |
| 階段 | 寬限期 / 攤還第 N 年 |
| 買房月繳 | 當期每月總繳金額 |
| 月繳本金 | 當期每月償還本金 |
| 月繳利息 | 當期每月利息 |
| 剩餘本金（萬） | 尚未償還的貸款餘額 |
| 當期月租金 | 含漲幅後的租金 |
| 每月投資額 |（買房月繳 + 持有成本）- 當期月租金（可為負值；正值為租房方投資額，負值為買房方投資額） |
| 租房方 ETF（萬） | 截至該年底的租房方 ETF 帳戶總值 |
| 買房方 ETF（萬） | 截至該年底的買房方 ETF 帳戶總值 |

## 五、互動行為

| 操作 | 行為 |
|------|------|
| 按「計算」 | 依輸入值計算結果，新增一列至比較表最下方 |
| 按「刪除」 | 移除該列（含展開的明細），序號不重新編排 |
| 按「展開明細」 | 在該列下方展開/收合每 5 年明細子表格（toggle） |
| 修改輸入欄位 | 不會影響已計算的結果列，僅影響下次計算 |
| 按「清除全部」 | 清空所有比較結果列與 LocalStorage（需二次確認） |
| 「欄位顯示」按鈕 | 比較總表與明細子表格各有獨立的欄位選擇器，可勾選欲顯示的欄位（預設全選） |
| Tab 切換「表格 / 圖表」 | 輸入區下方有 Tab 切換，可在表格檢視與圖表檢視之間切換 |

### 4.3 淨資產折線圖

在「圖表」Tab 下顯示：

| 項目 | 說明 |
|------|------|
| 圖表類型 | 折線圖（Recharts） |
| X 軸 | 年份（1 ~ 貸款年限） |
| Y 軸 | 淨資產（萬） |
| 線條 | 每方案產生兩條線：買房淨資產（實線）+ 租房淨資產（虛線） |
| 方案選擇 | Checkbox 勾選要顯示哪些方案（預設全選） |
| 色系 | 同方案同色系，以實/虛線區分買/租 |
| 互動 | hover 顯示數值 tooltip、Legend 點擊可隱藏線條 |

## 七、技術限制與排除項目

| 排除項目 | 原因 |
|----------|------|
| 後端 / 資料庫 | 只希望純前端實作，無需後端支援 |
| 多語系 | 僅繁體中文 |
| 登入 | 不需要，重新整理不會清空計算結果（已存入 LocalStorage） |
| 列印 / 匯出 | 此版本先不做 |

## 八、邊界條件與不考慮項目

| 不考慮項目 | 說明 |
|------------|------|
| 買房交易成本 | 契稅、印花稅、仲介費、代書費等不計入 |
| 租屋押金 | 押金機會成本不計入 |
| 利率變動 | 全程使用固定利率，不模擬浮動利率 |
| ETF 證交稅 | 賣出時 0.1%，影響極小，僅在 UI 備註提示 |
| ETF 配息所得稅 | 因人而異（綜所稅級距不同），使用者可自行調低 ETF 報酬率反映 |
| 二代健保補充保費 | 同上，視個人配息金額而定，不另行計算 |
| 房地合一土地漲價總數額扣除 | 實務可扣除但資料不易取得，不納入 |
| 持有成本年漲幅 | 管理費/維修費可能逐年調整，目前以固定月額簡化 |
